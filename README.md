# Student-Result
This project helps in simple result checking and basic data management
📘 Student Result Management System
📌 Project Name:
Student Result Management System (Python)

📖 Description:
This project is a simple Python application used to manage student marks and generate results automatically. It takes student name and marks of different subjects, then calculates total, percentage, and shows whether the student is PASS or FAIL.

🎯 Objective:
To automate student result calculation and reduce manual work in schools/colleges.

⚙️ Features:


Enter student name and marks


Subject-wise marks entry


Automatic total calculation


Percentage calculation


Pass/Fail result (35 rule)



🧮 Rule:


Marks ≥ 35 → PASS


Marks < 35 → FAIL



💻 Technologies Used:


Python (Core Programming)



▶️ How to Run:


Install Python


Open any IDE (VS Code / Jupyter Notebook)


Run the .py file


Enter student details and marks



📊 Sample Output:
Name: AmitTotal Marks: 340Percentage: 42.5Result: PASS
